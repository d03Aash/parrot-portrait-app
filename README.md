# parrot-portrait-app
Front-End Development Project | Bootstrap, HTML, CSS, JavaScript.

Responsive photo gallery using HTML, CSS, and Bootstrap.

Implemented interactive features: user photo uploads, attribute tagging (e.g., “Colorful”, “Friendly”), and a like counter.

Designed a dark-themed, mobile-friendly layout with Bootstrap’s grid system and components.

Demonstrated skills in form handling, dynamic content rendering, and UI/UX design principles.

Showcased creativity by blending playful design elements (emojis, tags) with professional responsiveness.
